```
tests/
├── __init__.py
├── conftest.py              # 统一测试配置

│   ├── test_algorithms.py   # 算法测试
│   └── test_auto_exposure.py # 自动曝光测试
├── integration/             # 集成测试
│   └── test_core.py         # 核心集成测试
├── e2e/                     # E2E 测试
│   └── test_frontend.py     # 前端 E2E 测试
└── fixtures/                # 测试夹具
```

### 测试运行命令

```bash
# 运行所有测试
python -m pytest tests/ -v

# 运行单元测试
python -m pytest tests/unit/ -v

# 运行集成测试
python -m pytest tests/integration/ -v

# 运行 E2E 测试
python -m pytest tests/e2e/ -v

# 运行覆盖率
python -m pytest tests/ --cov=backend --cov-report=html
```
