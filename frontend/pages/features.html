📋 特征说明

📍 峰位置特征（10 维）

检测每种矿物的特征峰位置，如石英 464 cm⁻¹、金刚石 1332 cm⁻¹

📊 峰强度特征（10 维）

特征峰的归一化强度（0-1），反映峰的相对强度

📏 峰宽度特征（10 维）

半高宽（FWHM），反映晶体质量和应力状态

⚖️ 强度比特征（5 维）

特征峰强度比值，如 I_1082/I_464，用于区分相似矿物

🌐 全局特征（5 维）

曲线下面积、光谱斜率、重心、偏度、峰度

✅ 筛选后特征（19 维）

通过方差阈值、相关性分析、RF 重要性筛选的最佳特征

📦 特征分布箱线图（按类别）

箱线图显示每个特征在不同矿物类别中的分布情况。箱体表示 25-75 百分位，中线表示中位数，须线表示最小/最大值。

🔗 特征相关性矩阵

热力图显示特征间的皮尔逊相关系数。红色表示正相关，蓝色表示负相关。高度相关的特征（|r|>0.9）会被筛选掉一个。

📊 特征重要性排序（随机森林 Top-15）

基于随机森林的 Gini 重要性排序。重要性越高，该特征对分类的贡献越大。Top-5 特征通常是石英 464cm⁻¹强度、金刚石 1332cm⁻¹强度等。

🎯 特征空间可视化（PCA 降维到 2D）

使用 PCA 将高维特征降维到 2D 平面，每个点代表一个样本，颜色代表矿物类别。类别间分离越清晰，说明特征区分度越好。
